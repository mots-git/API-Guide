Leads Missing

A client is having an issue where only some of their leads are not syncing from LinkedIn to their 3rd party, this is a different issue than if they're not syncing at all. If all of the leads are not syncing please follow this flow here.

First step is to ask the client to check their mapping settings, with some CRM systems if the mapping is not correct for all of the different fields this may mean that the lead may not sync to the 3rd party. In the CRM they have to map the different fields to each other, for example if the LGF has the fields below this is how they should be mapped:

LinkedIn CRM
First Name First Name
Last Name Last Name
Country Country
Work Email Work Email

If the client is unsure how to map the fields or check please send them the relavent setup which can be found here.

If the field which is not syncing needs a standardization value please send the client the follow sheet here. This contains the values for countries, states, job functions, etc which is needed for them to map correctly.
