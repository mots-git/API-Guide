Developer Access / CSE Team

If a client is having issues getting access to some of our APIs they made need MDP access. This is a portal which grants clients access to all of our public APIs. The only 2 this doesn't grant access to is LGF and Matched Audiences as these are private API connections. To get access to these the client needs to make a case to the CSE team and apply for approval.

Marketing Developer Platform (MDP) supports LMS API Products used by Developers and Partners, this can be found here.

Pre-sale Consultation: Managed Customers seeking consultation or direct access to APIs, pass them to CSE for build vs buy conversation.

Troubleshooting: If the client already built an app using our APIs, they are a developer. Direct them to our developer support team for troubleshooting requests.
Note: Client, NOT Sales will create a Developer support case.
